## 🏃프로젝트 소개
최근 대중들 사이에 러닝이 건강과 자기 관리의 중요한 trand로 자리 잡고 있으며, 기존의 러닝 앱과 다르게 러닝을 항상 즐겁고 지속 가능한 활동으로 만들기 위한 방법을 고민하다가 러닝 앱에 캐릭터 육성의 아이디어를 접목시켜 사용자의 동기부여를 줌으로써, 러닝을 오랫동안 지속 가능하게 하는 것을 목표로 개발하게 되었습니다.
